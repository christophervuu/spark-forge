```text
plans/tasks/P-###/T-###-##-slug.md
```

---

# Task Protocol

**Status Legend:**

- `[ ]` Not started | `[~]` In progress | `[x]` Complete | `[!]` Blocked

**Task Line Schema:**
`* [STATUS] [[FILENAME.md]](PATH/TO/FILE) - SHORT_DESCRIPTION`

Schema rules (canonical):

- `STATUS` must be one of: `[ ]`, `[~]`, `[x]`, `[!]`
- `FILENAME.md` must match the actual task filename
- `PATH/TO/FILE` must be a relative link to the task under `plans/tasks/`
- `SHORT_DESCRIPTION` must be a single-line summary (no extra punctuation fields)
- Do not embed full task specs here; details live only in the task file
